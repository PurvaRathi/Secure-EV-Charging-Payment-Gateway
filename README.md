Secure EV Charging Payment Gateway (Simulation)
Overview

This project is a simulated secure centralized EV charging payment gateway designed to demonstrate how modern cryptographic techniques can be applied to smart-grid payment systems.

The system integrates Blockchain, Lightweight Cryptography (ASCON), and Quantum Cryptography concepts to ensure secure, transparent, and efficient EV charging transactions.

 Features
 Secure payment processing using Lightweight Cryptography (ASCON)
 Blockchain-based ledger for immutable transaction records
 QR-based payment initiation for EV users
 User authentication using VMID & PIN
 End-to-end transaction flow simulation (User ↔ Kiosk ↔ Grid Authority)
 Simulation of Shor’s Algorithm to demonstrate vulnerabilities in classical cryptography
 System Architecture

The system simulates interactions between the following entities:

EV Owner (User Device)
Initiates charging session via QR scan and provides payment details
Charging Kiosk
Acts as an intermediary, encrypts data, and communicates with backend
Grid Authority (Central Server)
Validates transactions, manages accounts, and maintains blockchain ledger
Franchise (Charging Station Owner)
Receives payments upon successful transactions
 Workflow
User scans QR code generated by the charging kiosk
Enters VMID, PIN, and required charging amount
Data is securely transmitted to the backend
Grid Authority verifies credentials and balance
Transaction is recorded in the blockchain ledger
Charging session is approved/rejected accordingly
 Technologies Used
Blockchain – For secure and immutable transaction records
ASCON (Lightweight Cryptography) – For fast encryption on constrained systems
SHA-3 (Keccak-256) – For hashing IDs and transactions
Quantum Cryptography (Simulation) – Demonstrating RSA vulnerabilities using Shor’s Algorithm
