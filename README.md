# ***Secure EV Charging Payment Gateway (Simulation)***


### **Overview**

This project is a simulated secure centralized EV charging payment gateway designed to demonstrate how modern cryptographic techniques can be applied to smart-grid payment systems.

The system integrates Blockchain, Lightweight Cryptography (ASCON), and Quantum Cryptography concepts to ensure secure, transparent, and efficient EV charging transactions.

### **Features**
 1. Secure payment processing using Lightweight Cryptography (ASCON)
 2. Blockchain-based ledger for immutable transaction records
 3. QR-based payment initiation for EV users
 4. User authentication using VMID & PIN
 5. End-to-end transaction flow simulation (User ↔ Kiosk ↔ Grid Authority)
 6. Simulation of Shor’s Algorithm to demonstrate vulnerabilities in classical cryptography


### **System Architecture**
 The system simulates interactions between the following entities:
1. EV Owner (User Device) : Initiates charging session via QR scan and provides payment details
2. Charging Kiosk : Acts as an intermediary, encrypts data, and communicates with backend
3. Grid Authority (Central Server) : Validates transactions, manages accounts, and maintains blockchain ledger
4. Franchise (Charging Station Owner) : Receives payments upon successful transactions


### **Workflow**
- User scans QR code generated by the charging kiosk
- Enters VMID, PIN, and required charging amount
- Data is securely transmitted to the backend
- Grid Authority verifies credentials and balance
- Transaction is recorded in the blockchain ledger
- Charging session is approved/rejected accordingly


### **Technologies Used**
1. Blockchain – For secure and immutable transaction records
2. ASCON (Lightweight Cryptography) – For fast encryption on constrained systems
3. SHA-3 (Keccak-256) – For hashing IDs and transactions
4. Quantum Cryptography (Simulation) – Demonstrating RSA vulnerabilities using Shor’s Algorithm
